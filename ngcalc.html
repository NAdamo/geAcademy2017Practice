Calculator practice

Exercise

  1. Build a calculator application similar to this: https://codepen.io/FreeCodeCamp/full/rLJZrA/
  2. Create a new angular application
  3. Create a model for Operation. With the following properties:
    • firstOperand: number
    • secondOperand: number
    • operator: enum or string
  4. Create a model for CalculatorHistory. With the following properties:
    • operation: Operation
    • result: number
    • date: Date (default: new Date())
  5. Create a calculatorForm component. With the following instructions
    • It must contain an input for the firstOperand
    • It must contain a select for the operators.
    • Generate the option for the operators from an Array. The operations are the following:
      • Add
      • Substract
      • Multiple
      • Divide
      • Modulo
    • It must contain an input for the secondOperand
    • It must contain a calculate button.
    • The component must have an event which broadcasts an Operation object, when the form is submitted.
  6. Create a calculatorResult component. With the following instructions
    • It must have an input property for an Operation.
    • It must have a calculate method, which works with an Operation typed object.
    • It must have a delay method, which calls a setTimeout a few times before calls the calculate method.
    • It must show a loader while the result is calculating
    • It must show the result, when the calculation is done
  7. Create a calculateHistory component. With the following instructions:
    • It has an input property which accepts an Array of CalculatorHistory
    • It must show a table for history elements
  8. In the main component connect the components.
