Form practice

Exercise

  1. Create a new HTML file
  2. Create a new JavaScript file and attach to the HTML.
  3. Generate a form with at least 5 five number input.
  4. Generate two buttons within the form. The buttons will do a the following:
    • Add a new number input to the form.
    • Calculate a sum.
  5. Implement the the sum algorithm, which works the following:
    1. Create a new array
    2. Iterate through all input fields and get the value of the field and push it into the array
    3. Iterate through the array and create a new array where the items the sum of the item and previous and the next item. If previous or next item is undefined use 0 instead.
    4. Iterate through the new array and sum all of the items
